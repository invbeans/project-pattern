## Modules
- Backend
  - Java
  - Spring
- Frontend
  - JavaScript
  - TypeScript
  - Axios
  - Vue
  - Bootstrap
  - Bootstrap-Vue

## Requirements

- JDK 1.8
- Maven 3
- Node.js 14.17.5 (Optional)

## Build application

`mvn install`

## Run application

`mvn --projects backend spring-boot:run`

## Run tests

`mvn test`

## Environment variables:

- PORT
